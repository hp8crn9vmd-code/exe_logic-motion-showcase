# exe_logic Motion Showcase
A 100% code-generated 2D motion showcase with Swiss design aesthetics and
production-grade engineering.

### Features
- 10 unique motion modules
- System/user theme override modes
- Reduced motion support (WCAG 2.1 AA)
- Offscreen animation pausing
- Strict CSP compliance
- Full keyboard navigation

### Development
    npm install
    npm run dev

Production
    npm run build
    npm run preview

Testing
    npm test
    npm run test:e2e
    npm run verify
